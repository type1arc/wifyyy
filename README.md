# Purrito
A CLI program written in Rust which fetches N number of waifu images, saves them to disk and displays them in the terminal using chafa.

### install dependencies
```fish
sudo pacman -S chafa
```
### installation
```fish
#install rust
sudo pacman -S rust cargo rustc
```

```fish
# clone this repository
git clone https://github.com/type1arc/purrito && cd purrito
```

### usage
```fish
# build and run the program
cargo build && cargo run -- --count n
# replace n with any integer
# get n number of waifus
```
